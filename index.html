BlurChat - Privacy for WhatsApp Web

Last Updated: December 25, 2024

Overview

BlurChat is a browser extension that adds visual privacy protection to WhatsApp Web by applying CSS blur effects to chat content.

How the Extension Works

BlurChat works entirely through CSS style injection:

  1. When you visit WhatsApp Web, the extension injects CSS rules
  2. These CSS rules apply visual blur filters to selected elements
  3. Hovering reveals content temporarily via CSS :hover states
  4. No JavaScript reads, processes, or transmits your message content
